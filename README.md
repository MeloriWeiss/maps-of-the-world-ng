# Maps-of-the-World

Проект создан с помощью Angular CLI 19.2.14 + Nx.

## Запуск

Важно! Перед установкой убедитесь, что ваша версия node.js соответствует той, которая указана в `.nvmrc`.

Для установки зависимостей, используйте:

```bash
yarn install
```

### Разработка

Для запуска фронтенда в режиме разработки:

```bash
yarn start:web
```

Для запуска бэкенда в режиме разработки:

```bash
yarn start:api
```

Для запуска приложений, необходимых для работы бэкенда в режиме разработки (требуется `Docker`):

```bash
yarn start:api-deps:docker
```

### Сборка

Для сборки фронтенда:

```bash
yarn build:web:prod
```

Для сборки бэкенда:

```bash
yarn build:api:prod
```

Для сборки бэкенда и приложений, необходимых для его работы на сервере, в одном `docker`-образе:

```bash
yarn build:api:prod:docker
```

База данных поднимается на сервере отдельно, без Docker
